Site desenvolvido em HTML, CSS e JavaScript por Arthur Augusto Rodrigues Piratello de Castro.



O QUE É O SITE ?


O site é composto por 7 telas. A tela inicial é composta pelos botões Sobre e Contatos, um carrossel e algumas notícias. Ao clicar no botão Sobre o usuário é direcionado para uma página com algumas informações sobre o site, como ele foi feito e por quem. Já no botão Contatos, o usuário é direcionado para uma página com um formulário para preencher caso queira enviar alguma mensagem. No carrossel o usuário poderá escolher qual dos campeonatos ele mais se interessa (Brasileirão, Copa do Mundo, Libertadores e Champions League), cada um dos campeonatos tem sua própria tela.
